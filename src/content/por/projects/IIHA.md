O IIHA é um projeto realizado no **NICC** sobre o estudo da reincidência e das carreiras criminais a partir de arquivos digitais da justiça penal.

Contribuí para a proposta e para a orientação científica do projeto como **co-PI**. O trabalho baseou-se em dados do **Registo Criminal Central** e da base de dados das prisões.

O projeto desempenhou um papel importante no desenvolvimento de uma infraestrutura de dados mais robusta para o estudo das trajetórias penais.
